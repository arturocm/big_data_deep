Que es Big Data?

Para decir que tiene Big Data una empresa necesita:

  • Texto (2K páginas)
  • Social (5K seguidores, 10TW/posts x seguidor)

Que es Data Science?

Data Science es un campo interdisciplinario que involucra los procesos y sistemas para extraer conocimiento o un mejor entendimiento de grandes volúmenes de datos en sus diferentes formas y formatos.

Que es Machine Learning?

"Descubrimiento de funciones que partan el espacio de datos, ajusten a una curva sin conocer su orígen, o detecten grupos emergentes."

  • Dr. Fernando Esponda (ITAM)

Cual es el objetivo de Machine Learning?

Identificar patrones equivocándose lo menor posible.

Como lo logra?

Balanceando componentes del error: sesgo + varianza.

Algunos usos de ML - Clasificación

Clasificación: Partir el espacio de datos en N categorías con funciones.

Algunos usos de ML - Regresion

Regresion: Función que se ajuste al espacio de datos.

Clustering: Descubrimiento de grupos emergentes (No Supervisado)
